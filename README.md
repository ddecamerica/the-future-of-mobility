# The Future of Mobility

**Palm Beach County’s Connected Mobility Network**

This repository hosts the public image assets used to support the GIS StoryMap for **The Future of Mobility**, a DDEC LLC proposal prepared for the **2026 ULI West Palm Beach Forum**.

The StoryMap presents a connected mobility vision for Palm Beach County, beginning with the **Phase 1 Downtown West Palm Beach / Airport Connector**, an elevated automated people mover concept linking Palm Beach International Airport, downtown West Palm Beach, the Convention Center, CityPlace, Brightline, and nearby activity centers.

## Purpose

This repository is used as a lightweight public asset library for the StoryMap.

Images stored here can be referenced directly through GitHub Pages and embedded into ArcGIS StoryMaps or related web-based presentation tools. The goal is to keep all visual assets organized, stable, and easy to access during StoryMap development.

## Naming Convention

Use lowercase file names with hyphens.

Good:

```text
phase-01-downtown-wpb-airport-connector-route-map-01.png
pbc-stat-drive-alone-77pct-01.png
last-mile-av-zone-map-01.png
```

Avoid:

```text
Hialeah Cover Final.jpg
Map FINAL FINAL (2).png
Tamarind Ave. Top View.png
crash-analysis-map.jpg
community engagement image.png
```

## File Naming Rules

1. Use lowercase letters.
2. Use hyphens instead of spaces.
3. Avoid special characters, parentheses, accents, and unnecessary punctuation.
4. Include the project area or asset category when useful.
5. Use two-digit numbering when multiple versions of the same asset may exist.
6. Do not rename or move an image after its URL has been added to the StoryMap.

## Updating Assets

Once an image is used in the StoryMap, its URL should remain stable.

For minor visual corrections, replace the existing file using the same filename.

For major revisions, create a new versioned file and update the StoryMap link manually.

Example:

```text
phase-01-downtown-wpb-airport-connector-route-map-01.png
phase-01-downtown-wpb-airport-connector-route-map-02.png
```

## Project Context

The Future of Mobility proposes a connected transportation network for Palm Beach County, beginning with a Phase 1 connection between Palm Beach International Airport and downtown West Palm Beach.

The first phase focuses on a 5.4-mile elevated guideway with nine stations, connecting major destinations including PBI, the Convention Center, CityPlace, Brightline, and the downtown core.

The broader StoryMap presents the mobility challenge facing Palm Beach County, including regional growth, tourism demand, airport passenger activity, commute patterns, and the need for a more connected transportation network.
