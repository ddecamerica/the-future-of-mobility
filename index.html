🗺️ PBI–Downtown APM Connector

Storymap Assets Repository

Welcome to the storymap assets repository for the PBI–Downtown APM Connector project. This site hosts all media assets used in our GIS story map, including cover images, section graphics, maps, and project logos.

📍 Project Vision:

Planning, vision, and infrastructure behind the PBI–Downtown APM Connector: A connected, scalable, and financially sustainable transportation future for Palm Beach County.

📂 Asset Structure

  • storymap-assets/
    • assets/
      • images/
        • cover/ – Cover images
        • sections/ – Section-specific graphics
        • maps/ – Map visualizations
        • logos/ – Branding assets

🔗 How to Access Images

All assets are publicly available through GitHub Pages. Use these URLs in your GIS story map:

https://ddecamerica.github.io/the-future-of-mobility/storymap-assets/assets/images/[category]/[filename]

Example URLs:

  • Cover: ...images/cover/hialeah-cover-main.jpg
  • Section: ...images/sections/existing-conditions-01.jpg
  • Map: ...images/maps/crash-analysis-map.jpg
  • Logo: ...images/logos/project-logo.png

💡 Tip for GIS Story Maps:

When configuring your story map (e.g., in ArcGIS StoryMaps, Mapbox, or similar platforms), use the full URLs provided above to embed images directly from this repository. This ensures all assets remain centralized and easy to update.

🚀 Getting Started

  1. Browse the storymap-assets folder on GitHub
  2. Copy the asset URLs from the repository
  3. Paste them into your GIS story map platform
  4. Enjoy seamless image integration!
