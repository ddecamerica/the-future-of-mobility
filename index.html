Project context

One place for every StoryMap image link.

This repository is the public source for StoryMap media assets. The main proposal presents a five-phase mobility network for Palm Beach County, with Phase 1 focused on a Downtown West Palm Beach and airport connector. This page keeps image URLs clean, consistent, and easy to share with the GIS StoryMap team.

Phase 1 reference Downtown WPB and PBI Airport connector. 5.4 miles, 9 stations, elevated guideway concept.
Network role Connects airport, Convention Center, CityPlace, Brightline, NoRa, Palm Tran, Tri-Rail, Amtrak, and RideWPB.
Asset purpose Hosts images, maps, graphics, and brand assets used by the GIS StoryMap freelancer.

Folder structure

Keep the repo organized by use case.

The structure below avoids unrelated labels like Hialeah, crash analysis, or community engagement. It is specific to The Future of Mobility and the Palm Beach County APM concept.

assets/images/brand/

DDEC logos, project marks, and neutral brand assets.

assets/images/cover/

Hero images and opening visuals for the StoryMap.

assets/images/statistics/

Visual cards for population, visitors, airport passengers, and travel behavior.

assets/images/phase-01/

Route maps, station graphics, and visuals for the Downtown WPB and PBI connector.

assets/images/network/

Regional network integration graphics and transit connection diagrams.

assets/images/last-mile/

Autonomous vehicle zone graphics and first-mile or last-mile connection visuals.

Naming convention

Readable, stable, and StoryMap-ready.

File names should describe the project, asset type, location or subject, and sequence number. Once a file is live in the StoryMap, keep its name and folder location stable to avoid broken image links.

Use lowercase only. Example: phase-01-route-map-01.png
Use hyphens, not spaces. Avoid spaces, underscores, parentheses, and special characters.
Lead with the project or section. Example: future-mobility-cover-hero-01.jpg
Version intentionally. Use -v02 only when a new public URL is needed.

Important: After the freelancer places a link in the StoryMap, do not rename or move that image. Replace the file with the same name for small corrections, or create a new versioned file for major changes.
